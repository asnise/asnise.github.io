วิธีใช้งาน:

  • ใช้เครื่องมือนี้เพื่อสร้างบทสนทนาสำหรับตัวละครหนึ่งตัว
  • แต่ละบรรทัดจะมีหมายเลข (index) และช่องกรอกข้อความ
  • สามารถเพิ่มตัวเลือก (choice) ได้สูงสุด 3 ตัวเลือกต่อบรรทัด โดยแต่ละตัวเลือกจะมีข้อความและหมายเลขบรรทัดถัดไป
  • นำเข้าไฟล์ JSON เพื่อแก้ไขข้อมูลเดิม หรือบันทึกไฟล์ใหม่ด้วยปุ่ม Save
